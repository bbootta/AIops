# known_methodology_limits.md

방법론적 한계와 정량 결과를 단정적으로 사용하면 안 되는 케이스.

## 항목
- LDP (low default portfolio):
  - 표본 부도건수가 매우 적어 KS/AUC 신뢰구간이 넓다.
  - 다년 누적 표본 또는 베이지안 추정 사용 검토.

- 신규 모형 / 짧은 관측 기간:
  - 시계열 안정성 판단 불가.
  - backtesting 기간 누적 후 재검증.

- VIF / Condition Index:
  - 비선형 관계는 잡지 못한다.
  - 변수 간 관계는 별도 그래프/도메인 검토 필요.

- 잔차 정규성:
  - Jarque–Bera p-value만으로 모형 적합성 판단 금지.
  - 시계열 회귀에서는 자기상관 / 이분산성 추가 진단 필요.

- 시나리오 적용:
  - 단순 multiplier 적용은 실무 적용에 한계.
  - 정상성 변환과 시차 구조의 정합성을 별도 검토.

- PSI:
  - bin 정의에 매우 민감.
  - 비교 시점 간 등급 정의 변경이 있으면 PSI는 해석 불가.
