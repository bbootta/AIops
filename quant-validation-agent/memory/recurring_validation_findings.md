# recurring_validation_findings.md

반복적으로 발견되는 정량검증 이슈를 누적·정리한다.

## 양식
- 발견일자
- 모형 / 검증 구분
- 이슈 요약
- 근거 지표
- 원인 후보
- 권장 추가 확인
- 후속 조치

## 예시
- 2026-05-06 / 스코어링 / 등급 E의 default rate 100%로 관측됨 → 등급 E 표본 부족 가능성 → 등급 정의 또는 합산 기준 검토 권장
- 2026-05-06 / PD / 단기 관측 PD가 장기평균 PD보다 큼 → 경기 국면 영향 → backtesting 기간 확대 권장
- 2026-05-06 / PD multiplier / severe 시나리오 multiplier가 base보다 작은 케이스 1건 → 시나리오 적용 입력 오류 또는 모형 비선형성 → 입력 검증 및 회귀 적합성 재검토
