# REPORT_TEMPLATE

## 1. 목적

양적검증 팀 에이전트가 작성하는 검증보고서 초안 템플릿이다. 본 보고서는 검증 결과 초안이며, 조치 이행 관리 문서가 아니다. Yellow, Red, Gray 판정 후보가 있는 경우 별도 Action Notice를 참조한다.

## 2. 보고서 작성 통제

- LLM은 직접 수치 계산, 재계산, 추정 계산, 임계값 산출을 수행하지 않는다.
- 모든 수치는 계산엔진, 승인된 리포트, 공식 데이터 증적 출처를 표기한다.
- 계산엔진 결과 없이 정량 결론을 작성하지 않는다.
- 판정 라벨은 Green, Yellow, Red, Gray만 사용한다.
- Green은 최종 승인 또는 공식 조직의 승인 결정을 의미하지 않는다.
- Yellow / Red / Gray는 별도 Action Notice를 생성한다.

## 4. 금지 표현 예시

- “본 모델은 승인 가능하다.” 대신 “제공 증적 기준 Green 후보이나 최종 승인은 공식 조직 결정에 따른다.”
- “PD를 계산한 결과” 대신 “계산엔진 결과에 따르면”으로 출처를 명시한다.
- “문제 없음” 대신 “현재 제공된 증적 기준 중대 이슈 후보 미발견”으로 제한한다.

## 5. 운영상 가정

- 보고서는 초안이며 전자결재, 모델위원회, 리스크위원회 등 공식 승인 문서를 대체하지 않는다.
- 시각화가 필요한 경우 계산엔진 또는 BI 도구의 공식 출력물을 첨부한다.
