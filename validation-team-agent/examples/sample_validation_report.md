# 검증 보고서 초안 — 개인신용평가모형 v3.2

> 본 문서는 검증 보조 산출물 초안입니다. 최종 검증 의견과 외부 제출은 인간 검증자의
> 검토와 승인을 거쳐야 합니다.

## 1. 요약
개인신용평가모형 v3.2의 정기 검증 결과 변별력은 참고 임계를 충족하나, 운영
표본의 PSI가 "주의" 구간에 진입함. 등급 5의 단조성 위반 1건이 관측됨.

## 2. 검증 목적
연 1회 정기 검증. 신규 신청 평가에서의 사용 적정성 점검.

## 3. 입력 데이터 및 전제
- 데이터: `./samples/credit_score_sample.csv` (비식별 처리)
- 컬럼: customer_id, obs_date, score, target, grade, set
- 표본: 124,500건 (개발 80,000 / 운영 44,500), 부도 1,872건 (1.50%)
- 표본 기간: 2022-01 ~ 2024-12
- score 부호 규약: 높을수록 위험

## 4. 검증 방법
- 변별력: `tools/metric_ks_auc.calculate_ks`, `calculate_auc_gini`
- 안정성: `tools/metric_psi.calculate_psi(bins=10)`
- 등급분포: 등급별 표본 / 부도 / 부도율 / 단조성
- 캘리브레이션: `tools/metric_cdr_sdr.compare_cdr_sdr`
- 표본 적정성: `middleware/sample_size_guard.check_sample_size`

## 5. 주요 결과
| 항목 | 개발 | 운영 |
|---|---|---|
| KS | 0.41 | 0.36 |
| AUROC | 0.78 | 0.74 |
| Gini | 0.56 | 0.48 |
| PSI (개발 vs 운영) | — | 0.18 |

등급분포: 등급 1~10의 부도율 (대체로 단조 증가, 등급 5에서 등급 6 대비 0.05%p 역전 1회).

## 6. 이상 징후 및 원인 후보
- **운영 PSI 0.18 (주의)**: 운영 표본 분포의 우측 꼬리 두꺼워짐. 후순위/저신용
  유입 증가 가능성.
- **등급 5 단조성 위반**: 변수 X의 운영 분포 변화 가능성. 추가 확인 필요.

## 7. 한계와 리스크
- 운영 표본 부도 건수가 등급 1~3에서 30건 미만 → 등급별 캘리브레이션 신뢰구간 넓음.
- 외부 거시 변수 변화 (금리, 부동산)의 영향이 분리되지 않음.
- 본 보고서의 모든 수치는 비식별 샘플 기준이며, 실제 운영 데이터와는 다를 수 있음.

## 8. 검증 의견 초안
"본 검증 결과 모형의 변별력은 참고 임계를 충족하나, 운영 환경에서 안정성
지표가 주의 구간에 진입함. 등급 5의 단조성 위반과 함께 다음 정기 모니터링
시점까지 사후 점검을 강화할 것을 권고함. 본 의견은 초안이며 인간 검증자의
최종 검토와 승인 후에만 효력을 가짐."

## 9. 추가 확인 사항
- 운영 표본의 신청 채널별 분포 추이
- 변수 X의 정의 변경 여부 / 데이터 파이프라인 변경 여부
- 등급 1~3의 표본 수 보강 가능성
- 챌린저 (logistic) 비교 결과 별도 산출
