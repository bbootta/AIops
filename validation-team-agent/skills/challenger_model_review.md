# Skill — Challenger Model Review

## 목적
챌린저 모형의 필요성·구성·비교 적정성을 점검한다.

## 입력
- 본 모형의 변별력/안정성/캘리브레이션 결과
- 챌린저 모형 후보 (예: 단순 logistic, gradient boosting, 단순 룰 기반)
- 비교 표본 (개발/운영)

## 절차
1. 챌린저 도입 사유 점검
   - 기존 모형의 변별력 저하, 분포 변화, 운영환경 변화 등
2. 챌린저 모형 적합성
   - 표본 적정성, 변수 사용, 누수 점검
3. 비교 평가
   - KS / AUROC / Gini / PSI 비교
   - 등급분포 비교
   - 캘리브레이션 비교
4. 일반화 가능성
   - 시계열 안정성, 표본 외 검증

## 산출물
- 챌린저 도입 사유 / 적합성 평가
- 본 모형 vs 챌린저 비교표
- 일반화 가능성 평가
- 추가 확인 사항

## 금지
- 챌린저 모형 채택의 자동 확정
- 표본 외 일반화 결론을 강하게 단정

## 완료 기준
- 비교표가 변별력·안정성·캘리브레이션 전반을 포함하는가
- 한계가 명시되었는가
